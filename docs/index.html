RISE   is an integrated scripting environment for students, educators, artists and game designers who want to learn JavaScript game design in a simple way.


RISE   is focused on ease-of-entry into prototyping creative ideas with immediate feedback. No local servers, no text consoles, no browser plugins... write code, click, see the output.

RISE   is designed to work the same on Mac or Windows. Projects are uniformly organized, making JavaScript game design and interactive media easy to teach and easy to learn. Easy-Start templates are built in for many major frameworks, including Phaser, PixiJS, Canvas programming or vanilla JavaScript.

RISE   is open-source under the MIT License making it free and available for anyone to use or modify as needed. If you're interested in contributing to the project, check out the repository at Github.
