Get RISE

Mac Install Package: RISE-1.0.1.dmg
Open the install package and drag it to your Applications folder.

Windows Installer: RISE.Setup.1.0.1.exe
Double-Click for a standard Windows install

Example Projects: EXAMPLES.zip
Unzip this folder to your favorite documents location. It contains several "hello world" scripts for various frameworks.

The source code and install instructions are also available at Github:
